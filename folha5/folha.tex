Vamos representar pontos de um plano cartesiano com duas coordernadas e regiões desse plano

\begin{lstlisting}[language=Haskell]

\item Considere a representação de uma relação binária nos inteiros como uma lista de pares.

\texttt{type Rel = [(Int , Int)]}

\begin{enumerate}

\item Escreva uma função \texttt{reflexiva :: [Int]->Rel -> Bool} que verifique se uma relação R em A é
reflexiva (R é reflexiva se e somente se $\forall x \in A , (x, x) \in R $)

Exemplos:
\begin{lstlisting}[language=Haskell]
reflexiva [1,2,3] [(1, 1), (2, 2), (1, 2),(3, 3)] = True
reflexiva [1,2,3] [(1, 2), (2, 3)] = False

reflexiva conj rel = and [ elem (x,x) rel | x <- conj ]

\end{lstlisting}


\item Escreva uma função \texttt{simetrica :: Rel -> Bool} que verifique se uma relação é
transitiva (R é transitiva se e somente se $\forall x,y, (x, y) \in R  \Rightarrow (y, x) \in R$).

Exemplos:
\begin{lstlisting}[language=Haskell]
simetrica [(1, 3), (3, 1), (2, 2)] = True
simetrica [(1, 2), (2, 3)] = False

simetrica [] == True
simetrica [(1,2),(2,1)] == True
simetrica [(1,2),(2,1),(1,3)] == False
simetrica [(1,2),(2,1),(1,3),(3,1)] == True
\end{lstlisting}

\item Escreva uma função \texttt{transitiva :: Rel -> Bool} que verifique se uma relação é
transitiva (R é transitiva se e somente se $\forall x,y,z,  (x,y) \in R \wedge (y,z) \in R \to (x,z) \in R$).

Exemplos:
\begin{lstlisting}[language=Haskell]
transitiva [(1, 3), (1, 2), (2, 3)] = True
transitiva [(1, 2), (2, 3)] = False

transitiva [(1,2),(2,4),(1,4)] == True
transitiva [(1,2),(2,4),(1,4),(2,3)] == False
transitiva [] == True
\end{lstlisting}
\end{enumerate}

Um mobile é uma estrutura constituída por uma haste, a partir
da qual objetos ponderados ou outras hastes são penduradas.

A figura abaixo ilustra um mobile. É uma haste suspensa por uma corda, com objetos pendurados
de cada lado. Também pode ser visto como uma espécie de alavanca
com o ponto de apoio sendo o ponto em que a haste está suspensa
pela corda. Pelo princípio da alavanca, sabemos que equilibrar um móvel
temos que o produto do peso dos objetos pela distância ao ponto de apoio
devem ser iguais, ou seja, $Wl \times Dl = Wr \times Dr$ onde
Dl é a distância da esquerda, Dr é a distância da direita, WL é o peso da esquerda e Wr é o peso da direita.

\begin{lstlisting}[language=Haskell]
import Data.List

data Prop =
    Const Bool
    | Var Char
    | Neg Prop
    | Conj Prop Prop
    | Disj Prop Prop
    | Impl Prop Prop
    deriving (Eq, Show)

prop1 = Impl (Var 'P') (Var 'Q')
prop2 = Impl (Neg (Var 'P')) (Var 'Q')
prop3 = Impl prop1 prop2


variaveis :: Prop -> [Char]
variaveis (Const x) = []
variaveis (Var x)   = [x]
variaveis (Neg p)   = variaveis p
variaveis (Conj p q) = nub ( variaveis p ++ variaveis q )
variaveis (Disj p q) = nub ( variaveis p ++ variaveis q )
variaveis (Impl p q) = nub ( variaveis p ++ variaveis q )

\end{lstlisting}

\item Escreva uma definição recursiva da função \texttt{removeImpl :: Prop -> Prop} que obtém
uma proposição equivalente removendo a implicação. A implicação pode ser removida usando a seguinte regra de equivalência:

\begin{equation}
p \to q \Leftrightarrow \neg p \vee q 
\end{equation}

\begin{lstlisting}[language=Haskell]
removeImpl prop1 == Disj (Neg (Var 'P')) (Var 'Q')
removeImpl prop2 == Disj (Neg (Neg (Var 'P'))) (Var 'Q')
removeImpl prop3 == Disj (Neg (Disj (Neg (Var 'P')) (Var 'Q'))) 
(Disj (Neg (Neg (Var 'P'))) (Var 'Q'))
\end{lstlisting}


\item Escreva uma definição recursiva da função \texttt{dual :: Prop -> Prop} que obtém
o dual duma proposição, i.e. a proposição que resulta de substituir todas as conjunções por disjunções (e vice-versa) e as constantes True por False (e viceversa); as variáveis e negações são inalteradas. Exemplos:
\begin{lstlisting}[language=Haskell]
dual (Neg (Var 'a'))
= Neg (Var 'a')
dual (Disj (Var 'x') (Neg (Var 'x')))
= Conj (Var 'x') (Neg (Var 'x'))
dual (Conj (Var 'a') (Disj (Var 'b') (Const False)))
= Disj (Var 'a') (Conj (Var 'b') (Const True))
\end{lstlisting}

\item Escreva uma definição recursiva da função \texttt{duplaNegacao :: Prop -> Prop} que obtém

\begin{equation}
\neg \neg p \Leftrightarrow p
\end{equation}

\begin{lstlisting}[language=Haskell]
duplaNegacao $ removeImpl prop1 == Disj (Neg (Var 'P')) (Var 'Q')
duplaNegacao $ removeImpl prop2 == Disj (Var 'P') (Var 'Q')
duplaNegacao $ removeImpl prop3 == Disj (Neg (Disj (Neg (Var 'P')) (Var 'Q')))

\end{lstlisting}
